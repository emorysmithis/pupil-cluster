# pupil-cluster
Unsupervised model to classify task-evoked pupil diameters

1. clone repo 
2. remove Experiment 1a from Eye data directory 
3. run script like this: 
```
python .\load_data.py <path to data directory> <data file extension>
```
```
python .\load_data.py '..\..\WM study\WM study\Eye data\' .gazedata
```
